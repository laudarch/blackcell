B.Math.Plane()

Represents a plane.

To create the object use B.Math.makePlane().

CONSTANTS:

{B.Math.Plane} X = {normal: B.Math.Vector3.X, distance: 0}

Positive direction along X-axis.

{B.Math.Plane} Y = {normal: B.Math.Vector3.Y, distance: 0}

Positive direction along Y-axis.

{B.Math.Plane} Z = {normal: B.Math.Vector3.Z, distance: 0}

Positive direction along Z-axis.

{B.Math.Plane} N_X = {normal: B.Math.Vector3.N_X, distance: 0}

Negative direction along X-axis.

{B.Math.Plane} N_Y = {normal: B.Math.Vector3.N_Y, distance: 0}

Negative direction along Y-axis.

{B.Math.Plane} N_Z = {normal: B.Math.Vector3.N_Z, distance: 0}

Negative direction along Z-axis.

PROPERTIES:

{B.Math.Vector3} normal

Normal of the plane.

{number} distance

Distance to the plane along the normal.
