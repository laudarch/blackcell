B.Math.Vector4()

Represents a 4D vector.

To create the object use B.Math.makeVector4().

CONSTANTS:

{B.Math.Vector4} ZERO = {x: 0.0, y: 0.0, z: 0.0, w: 0.0}

Zero vector.

PROPERTIES:

{number} x

{number} y

Y component.

{number} z

Z component.

{number} w
