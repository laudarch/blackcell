B.Math.Triangle()

Represents a triangle.

To create the object use B.Math.makeTriangle().

PROPERTIES:

{B.Math.Vector3} a

The first point.

{B.Math.Vector3} b

The second point.

{B.Math.Vector3} c

The third point.

METHODS:

Clones this triangle to a new triangle.
